# Koreanize

Turns romanized input text into korean characters and copies text to clipboard.

Romanization is based on the following key:

gg - ᄁ

n - ᄂ

d - ᄃ

dd - ᄄ

l/r - ᄅ

m - ᄆ

b - ᄇ

bb/pp - ᄈ

s - ᄉ

ss - ᄊ

ng - ᄋ

j - ᄌ

jj - ᄍ

ch - ᄎ

k - ᄏ

t - ᄐ

p - ᄑ

h - ᄒ

a -ᅡ

ae -ᅢ

ya -ᅣ

yae -ᅤ

eo/ou -ᅥ

e -ᅦ

yeo/ou -ᅧ

ye -ᅨ

o - ᅩ

wa - ᅪ

wae - ᅫ

oi - ᅬ

yo - ᅭ

u or oo - ᅮ

weo - ᅯ

we - ᅰ

wi/wee - ᅱ

yu or yoo - ᅲ

eu - ᅳ

eui - ᅴ

i/ee - ᅵ
